# 🤖 AI Study Assistant  
### 🎓 Student Dashboard & Intelligent Learning Resources Interface  

---

## 🚀 Overview

AI Study Assistant is a fully responsive frontend web application designed to centralize academic tools and study workflows into a single intuitive dashboard.

The platform provides structured access to assignments, documents, flashcards, quizzes, scheduling tools, and authentication interfaces — all built using semantic HTML5, modern CSS3, and Bootstrap 5.3.8.

This project simulates the frontend layer of a scalable EdTech platform.

---

## 🧠 Business Context

Students often deal with fragmented tools and scattered resources.

AI Study Assistant solves this by:

- 📚 Aggregating study materials  
- 🗂 Organizing assignments and deadlines  
- 🃏 Providing interactive revision tools  
- 📊 Offering a clean, mobile-first dashboard  

The goal is to simulate a lightweight Learning Management Interface (LMI) with production-level UI structure.

## 🎨 Design System

### 🎨 Color Palette

- Primary: `#20B486`  
- Primary Soft: `#eaf7f3`  
- Text Dark: `#111827`  
- Text Muted: `#8a8a8a`  
- Background Page: `#f7f9fb`  
- Surface White: `#ffffff`  

All colors are controlled via CSS variables for consistency and scalability.

### 🔤 Typography

- System font stack for cross-device consistency  
- Clear heading hierarchy (H1–H4)  
- 16px base font size for readability  

### 🧩 Layout Strategy

- 📱 Mobile-first responsive grid  
- 🗃 Modular dashboard card system  
- 🧭 Optional sidebar navigation (desktop view)  
- 🧱 Page-isolated structure for scalability

## ✨ Core Features

### 🏠 Dashboard  
Card-based overview of assignments, quiz progress, documents, and flashcards.

### 📌 Assignments  
Displays due dates, status indicators, and course metadata with visual prioritization.

### 📂 Documents Library  
Structured card interface for browsing and accessing study materials.

### 🃏 Flashcards  
Deck-based interactive flashcards supporting active recall learning.

### 📝 Quizzes  
Multiple-choice interface with scoring system and retake functionality.

### 📅 Schedule  
Visual layout for managing classes, study sessions, and deadlines.

### 🔐 Authentication  
Sign-in and registration forms with accessible and responsive UI structure.

---

## 📱 Responsive Design

Built with mobile-first principles:

- 📲 Small screens → Single-column layout  
- 💻 Medium screens → Two-column layout  
- 🖥 Large screens → Multi-column dashboard with full navigation  

Bootstrap breakpoints and custom media queries ensure smooth scaling across devices.
